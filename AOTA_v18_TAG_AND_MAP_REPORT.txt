Ashes of the Armistice v0.18 - Tag and Ownership Cleanup

What this pass does:
- Applies the ownership audit to colonial and regional start states.
- Replaces leftover KR-only start owners/controllers from the audit areas: NFA, MAF, GEA, PRF, HND, PAL.
- Cleans mixed tag leftovers inside overridden state files by removing undefined KR/vanilla-only core/claim tags that this mod does not define.
- Keeps Germany as a partial colonial survivor instead of a giant overseas hegemon.
- Unifies India under the Raj at game start.
- Splits New Zealand from Australia at game start.
- Keeps the Ottomans in the Levant/Mesopotamian core while placing Sinai/Suez under Egypt.
- Adds explicit mod tag definitions, colors, and history files for ALG, COG, IRE, FIN, and UKR so those focus trees/setup files are no longer orphaned.

Important notes:
- This pass focuses on start ownership and tag consistency, not deep province remapping.
- MAL, INS, RAJ, AST, NZL, EGY, and TUR are treated as standard playable tags in the mod setup.
- I have not boot-verified the build in HOI4 here; this is a file-level cleanup pass.
