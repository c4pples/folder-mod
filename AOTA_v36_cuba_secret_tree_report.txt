AOTA v36 Cuba Secret Ancap Tree

What this pass adds:
- A brand-new Cuba focus tree selected for CUB
- A secret anarcho-capitalist / corporate-nightmare branch hidden behind an event-gated decision
- Cuba-only decision:
  "Meet the Corporate Exiles"
- Cuba event:
  the player or AI chooses whether to admit American corporate personnel and their retainers
- If accepted:
  - sets AOTA_CUB_ANCAP_PATH_OPEN
  - unlocks the hidden focus branch
  - shifts Cuba toward market_radicalism
- If rejected:
  - the secret path remains closed

Design intent:
- Cuba becomes a dark Caribbean refuge for American corporate-political collapse
- The branch is intentionally dystopian: charter cities, private arbitration, payroll battalions, extraterritorial enclaves, and boardroom sovereignty
- The path is not available by default and must be consciously chosen through the event

Notes:
- This is a flavor/mechanics addition and has not been engine-tested here
- No base Cuban content existed in the package, so this pass adds a full custom Cuba tree from scratch
